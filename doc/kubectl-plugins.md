## Usage

1. Put 'kubectl-kube-top' file under one of your 'PATH' directories or add current location of 'kubectl-kube-top' file to your 'PATH' variable.
2. Check that plugin is available using `kubectl plugin list` command.
3. Plugin usage: `kubectl kube top [resource-name namespace-name]`. Default resource type: 'pod'. Default namespace name: 'default'.
